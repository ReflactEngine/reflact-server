# Reflact Server

The standalone server application for Reflact, powered by Minestom. It initializes the `ReflactEngine`, loads the world, and manages player connections.

## Tech Stack

*   **Framework:** Minestom (Snapshot 2026.01.08)
*   **Java:** 25

## Running

Start the server on port 25565:

```bash
./gradlew run
```
